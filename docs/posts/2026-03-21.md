## AI 资讯

**1. GPT-5.4 Thinking 成为 OpenAI 旗舰模型**
OpenAI 的 GPT-5.4 Thinking 正式成为其旗舰模型，融合了增强推理、编程与 Agentic 工作流，支持高达 100 万 Token 上下文，幻觉率显著降低。与 Claude 4.6 和 Gemini 3.1 Pro 并列，代表了新一代通用大模型的高水位线。
🔗 [Renovate QR](https://renovateqr.com/blog/ai-model-releases-2026)

**2. Meta 发布 Llama 4：原生多模态，最高1000万 Token 上下文**
Meta 发布 Llama 4（含 Scout 和 Maverick 两个变体），基于 MoE 架构，原生支持文本、图片与短视频处理。Scout 变体以高达 1000 万 Token 的上下文窗口创下开源模型新纪录。
🔗 [LLM Stats](https://llm-stats.com/llm-updates)

**3. DeepSeek V4 发布：万亿参数，推理速度提升1.8倍**
DeepSeek V4 带来四项核心创新：分层 KV 缓存（降低内存40%）、Sparse FP8 解码（推理加速1.8倍）、优化预训练课程、以及支持100万+ Token 检索的 Engram 架构，进一步巩固其在开源 AI 赛道的领先地位。
🔗 [Mule AI Blog](https://muleai.io/blog/2026-03-03-deepseek-v4-open-source-ai-revolution/)

**4. xAI Grok 4.20：四智能体并行处理每个查询**
Grok 4.20 采用独特的四智能体并行架构：Grok 负责协调，Harper 实时事实核查（接入 X 平台数据），Benjamin 处理逻辑与编程，Lucas 负责创意推理，四个智能体相互辩论后合成最终答案。
🔗 [Labla.org](https://www.labla.org/latest-ai-model-releases-past-24-hours/ai-model-releases-march-17-2026-claude-gets-memory-microsoft-pulls-back-and-the-race-continues/)

**5. Google Gemini 付费用户同比增长258%**
Google Gemini 付费用户年增长率高达258%，超越 Claude 的200%。Google 还宣布将向五角大楼300万人的非涉密工作部署 Gemini AI 助理，进一步扩大企业版图。
🔗 [UC Strategies](https://ucstrategies.com/news/googles-ai-grew-258-while-openai-and-anthropic-fought-in-court/)

**6. Anthropic 起诉美国国防部，业界联署声援**

🔗 [TechCrunch](https://techcrunch.com/2026/03/09/openai-and-google-employees-rush-to-anthropics-defense-in-dod-lawsuit/)

**7. Anthropic 成立社会 AI 研究院**
Anthropic 宣布成立 Anthropic Institute，专注研究先进 AI 在经济、社会和安全层面的影响，标志着其从单纯的模型研发向政策相关研究延伸，呼应全球 AI 监管加速趋势。
🔗 [Medium - AI Pulse](https://danielquinteros.medium.com/google-openai-anthropic-updates-edc94fc6f10e)

**8. 美国通过《AI 问责法》，要求定期发布偏见审计报告**
美国于2026年3月正式颁布《AI 问责法》，要求在重要决策中使用 AI 的企业定期开展并公开偏见审计。华盛顿州同步通过五项 AI 相关法案，涵盖信息披露、未成年人安全、医保 AI、深度伪造和数字肖像权。
🔗 [Transparency Coalition](https://www.transparencycoalition.ai/news/ai-legislative-update-march20-2026)

**9. AI 驱动广告市场2026年预计达570亿美元**

🔗 [Julian Goldie](https://juliangoldie.com/ai-news-march-2026/)

🔗 [NeuralBuddies](https://www.neuralbuddies.com/p/ai-news-recap-march-20-2026)

## 前端资讯

**1. Next.js 16.2 发布：开发启动速度提升400%，支持 AI 智能体工作流**
Next.js 16.2（3月18日）带来约400%的 `next dev` 启动提速、约50%的渲染提速，以及200+项 Turbopack Bug 修复。新增 `AGENTS.md` 脚手架、浏览器日志转发和实验性 Agent DevTools，迈向 AI 原生开发工作流。
🔗 [Next.js 官方博客](https://nextjs.org/blog/next-16-2)

**2. Turbopack 新特性：SRI 支持、Server Fast Refresh、Web Worker Origin**
Next.js 16.2 的 Turbopack 专项更新新增 Subresource Integrity（SRI）支持、`postcss.config.ts` 支持、Server Fast Refresh 和 Web Worker Origin 支持（适用于 WASM 库）。实际项目编译速度提升400–900%，应用刷新速度提升67–100%。
🔗 [Next.js Turbopack 博客](https://nextjs.org/blog/next-16-2-turbopack)

**3. TypeScript 6.0 正式 GA：最后一个 JS 编译器版本，ES5 支持弃用**
TypeScript 6.0（约3月17日 GA）是基于 JavaScript 编译器代码库的最后一个版本，TypeScript 7.0 将切换至全新 Go 语言编译器。重大变更：ES5 作为编译目标已弃用，推荐 ES2015+。
🔗 [TypeScript 官方博客](https://devblogs.microsoft.com/typescript/announcing-typescript-6-0-rc/)

**4. Svelte 3月更新：滚动感知页面过渡动画**
Svelte 发布3月月度更新，带来可水化 CSP 注入脚本修复，以及导航回调携带滚动位置信息，使基于滚动位置的页面过渡动画成为可能。Svelte 在 State of JS 2025 中开发者满意度继续领跑。
🔗 [Svelte 官方博客](https://svelte.dev/blog/whats-new-in-svelte-march-2026)

**5. SvelteKit 2.54.0 发布，适配 vite-plugin-svelte 7**
SvelteKit 2.54.0（3月11日）在 `enhanced-img` 包中新增对 `vite-plugin-svelte` 7 的支持，使 Svelte 生态与最新 Vite 工具链对齐。`@sveltejs/vite-plugin-svelte` 7.0.0 成为当前主版本。
🔗 [SvelteKit GitHub Releases](https://github.com/sveltejs/kit/releases)

**6. Angular 21.2.5 发布：Zoneless 变更检测成为默认**
Angular 21.2.5（约3月18-19日）正式将 Zoneless 变更检测设为默认，彻底移除 Zone.js 依赖，减小包体积、提升性能，并实现更可预测的渲染行为。
🔗 [VersionLog](https://versionlog.com/angular/21.0/)

**7. Chrome 切换为两周发布周期，加速 Web 标准落地**
Google 宣布（3月3日）Chrome 从四周发布周期改为两周，Chrome 146 是新节奏下首批稳定版之一。更频繁的发布节奏将加快 Web 标准特性落地速度。
🔗 [9to5Google](https://9to5google.com/2026/03/03/chrome-two-week-updates/)

**8. Interop 2026 宣布：跨文档视图过渡与滚动动画为优先目标**
WebKit 与各大浏览器厂商发布 Interop 2026 重点方向，跨文档（MPA）视图过渡和滚动驱动动画被列为跨浏览器互操作优先级。Chrome 145 已落地基于滚动偏移阈值触发时间动画，无需 JavaScript。
🔗 [WebKit 官方博客](https://webkit.org/blog/17818/announcing-interop-2026/)

**9. W3C 发布 CSS Snapshot 2026：Masonry 布局、corner-shape 等进入稳定**
W3C 发布 CSS Snapshot 2026，汇总当前稳定 CSS 特性。新晋"生产可用"特性包括：原生 CSS Masonry 布局（`display: masonry`）、`sibling-index()` / `sibling-count()` 函数、`corner-shape` 几何边框、网格/弹性布局间距装饰，以及可样式化的 `<select>`（`appearance: base-select`）。
🔗 [W3C](https://www.w3.org/TR/css-2026/)

**10. Next.js 16.2 AI 专项改进：AGENTS.md 与浏览器日志转发**
Vercel 专文介绍 Next.js 16.2 的 AI 向改进：新增 `AGENTS.md` 文件帮助 AI 智能体理解项目结构，浏览器日志转发支持 AI 驱动的调试工作流，以及开发服务器锁文件处理机制。
🔗 [Next.js AI 博客](https://nextjs.org/blog/next-16-2-ai)
