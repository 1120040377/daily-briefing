## 前端资讯

**1. State of JS 2025：TypeScript 独占使用率升至 40%**
State of JS 2025 年度调查显示，40% 的受访开发者完全使用 TypeScript 编写代码（2024 年为 34%），超过 80% 的前端职位要求 TypeScript，已成为专业开发默认标准。
🔗 [InfoQ](https://www.infoq.com/news/2026/03/state-of-js-survey-2025/)

Next.js 16.2 于 3 月 18 日发布，新增 agent-ready create-next-app、浏览器日志转发、实验性 Agent DevTools，启动速度提升约 400%，渲染速度提升约 50%。
🔗 [Releasebot](https://releasebot.io/updates/vercel/next-js)

**3. Vite 8 正式发布：Rolldown 集成，构建速度飞跃**
Vite 8 实验性引入基于 Rolldown 的 Full Bundle Mode，开发服务器启动速度提升 3 倍，完整重载提升 40%，网络请求减少 10 倍，内置 TypeScript emitDecoratorMetadata 支持。

**4. React 19.2：View Transitions、Activity 正式可用**
React 19.2 带来 View Transitions 和 Activity API 正式支持，React Compiler 持续普及，手动 useMemo/useCallback 已被视为过时写法。

**5. Svelte 3 月更新：createContext、HTML 注释、Laravel Starter Kit**
Svelte 官方 3 月更新包含程序化 createContext API、HTML 标签内注释支持、better-auth 官方插件，以及 Laravel + Svelte 官方 Starter Kit。State of JS 2025 中 Svelte 蝉联满意度榜首。

**6. Vue 3.6 Vapor Mode 极速挂载 Demo 发布**
Vue 3.6 Vapor Mode 可完全绕过虚拟 DOM，生成接近手写优化代码的输出，在挂载速度和响应式性能上取得显著提升，计划 2026 年正式推出。

Tailwind CSS v4.2.0 新增官方 @tailwindcss/webpack 插件、四种新默认色板（mauve/olive/mist/taupe）、完整的逻辑属性工具类和 font-features 工具。

**8. Astro 启动 Rust/Oxc 编译器研发**
Astro 团队宣布正在基于 Rust 和 Oxc 工具链研发新一代编译器，目标大幅提升静态站点生成性能，追赶 Vite 8 的 Rolldown 集成趋势。
