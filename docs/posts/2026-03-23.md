## AI 资讯

**1. OpenAI 推出 GPT-5.4 mini，向免费用户开放"思考"功能**
GPT-5.4 mini 正式在 ChatGPT 中向 Free 和 Go 用户通过"Thinking"功能提供服务，也作为 GPT-5.4 Thinking 的流量限制兜底模型。GPT-5.1 系列已于 3 月 11 日正式下线，现有对话自动迁移至对应新版本。
🔗 [Meyka](https://meyka.com/blog/openai-march-22-gpt-52-adds-aug2025-cutoff-new-enterprise-connectors-2203/)

**2. OpenAI GPT-5.2 知识截止日期延伸至2025年8月，新增企业连接器**
OpenAI 于 3 月 22 日更新了 GPT-5.2，将知识截止日期从此前版本延伸至 2025 年 8 月，并为 ChatGPT Business 增加新的企业级数据连接器。Legacy 深度研究模式将于 3 月 26 日停用。
🔗 [Releasebot](https://releasebot.io/updates/openai/chatgpt)

**3. Anthropic 3月提升全线用户使用配额，Claude Sonnet 4.6 登场**
Anthropic 宣布 3 月期间全面提升免费及付费用户的消息使用上限。Claude Sonnet 4.6 同步发布，在编程、计算机操作、长上下文推理和 Agent 规划等方面全面升级，并带来 100 万 Token 上下文窗口 Beta。
🔗 [Releasebot](https://releasebot.io/updates/anthropic/claude)

**4. Anthropic 推出 Claude Code 新功能：--channels 研究预览与插件市场**
Claude Code 本月更新引入 --channels 参数支持 MCP 消息推送，新增 --bare 脚本调用模式，以及更丰富的状态栏与插件控制功能。Anthropic 同步上线面向 Team/Enterprise 的插件市场和管理员控制台。
🔗 [Claude Code Changelog](https://code.claude.com/docs/en/changelog)

**5. Google Gemini 个人智能功能向全美免费开放**
Google 于 3 月 17 日将 Gemini 个人智能功能（Personal Intelligence）向全美免费用户开放，允许 Gemini 访问 Gmail、相册、Google Docs 和 YouTube 历史记录，此前该功能仅限付费高级用户。
🔗 [Roborhythms](https://www.roborhythms.com/google-gemini-personal-intelligence-free-gmail-march-2026/)

**6. Google Gemini 深度集成 Workspace：Docs、Sheets、Drive 全面升级**
Gemini 新增 Workspace 自动化能力，包括：在 Sheets 中通过自然语言从零构建表格并自动从 Gmail 提取数据；在 Docs 中跨 Drive 文件合成内容；在 Drive 搜索中提供 AI 概述摘要。功能首先向 AI Ultra 和 Pro 用户推出 Beta。
🔗 [Google Blog](https://blog.google/products-and-platforms/products/workspace/gemini-workspace-updates-march-2026/)

**7. xAI 完成超额 $200 亿 E 轮融资，Grok 4.20 Beta 开放企业 API**
xAI 完成超额 E 轮融资，募资金额达到 200 亿美元（目标为 150 亿），持续扩张 Colossus 超算集群规模。同时 Grok 4.20 Multi-agent Beta 和 Grok Imagine 统一视音频生成 API 正式向企业用户开放。
🔗 [xAI News](https://x.ai/news)

**8. xAI 招募 Mistral 联合创始人，全力押注超级智能研究**
据 FinTech Weekly 报道，机器人学研究员、Mistral AI 联合创始人 Devendra Singh Chaplot 以及 Thinking Machines Lab 创始成员加入 xAI 及 SpaceX，直接参与 Grok 模型训练。
🔗 [FinTech Weekly](https://www.fintechweekly.com/news/xai-superintelligence-devendra-chaplot-mistral-thinking-machines-lab-grok-march-2026)

**9. Perplexity 重磅更新：Model Council 让三大模型同时协作答题**
Perplexity Max 订阅用户现可在 Perplexity Computer 中使用 Model Council 功能，让 GPT-5.4、Claude Opus 4.6、Gemini 3.1 Pro 三个顶级模型同时运行并共同回答问题。Deep Research 功能同步升级，现运行于 Opus 4.5。
🔗 [The Agency Journal](https://theagencyjournal.com/whats-new-in-perplexity-this-march-multi-model-magic-smarter-research-and-ai-that-actually-listens/)

**10. Yann LeCun 创立 AMI Labs 完成 $10.3 亿种子轮融资**
Advanced Machine Intelligence（AMI）Labs 由 Yann LeCun 创立，完成欧洲史上最大种子轮融资，金额达 10.3 亿美元，估值 35 亿美元。与此同时，美国能源部宣布拨款 2.93 亿美元用于 AI 驱动的国家科学研究项目（Genesis Mission）。
🔗 [AI Flash Report](https://aiflashreport.com/model-releases)

## 前端资讯

**1. Next.js 16.2 发布：启动速度提升 400%，AI Agent 工具链集成**
Next.js 16.2 于 3 月 18 日发布，带来基于 Turbopack 的服务端 Fast Refresh、~400% 更快的 next dev 启动速度与 ~50% 更快的渲染速度，并引入 Agent-ready 脚手架与浏览器日志转发功能。
🔗 [Releasebot](https://releasebot.io/updates/vercel/next-js)

**2. Vite 8 正式发布：Rolldown 全量模式实验性集成，内置 TS Decorator 支持**
Vite 8 要求 Node.js 20.19+ / 22.12+，实验性引入基于 Rolldown 的 Full Bundle Mode，开发服务器启动速度提升 3 倍，完整重载提升 40%，网络请求减少 10 倍；同时内置 TypeScript emitDecoratorMetadata 支持。
🔗 [Vite Blog](https://vite.dev/blog/announcing-vite8)

**3. Vue 3.6 Vapor Mode：彻底抛弃虚拟 DOM，渲染性能大幅跃升**
Vue 3.6 中的 Vapor Mode 选项可完全绕过虚拟 DOM，生成接近手写优化代码的输出，在挂载速度和响应式性能上取得显著提升，Vue 3 已成为社区全面拥抱的现代默认标准。
🔗 [Nucamp Blog](https://www.nucamp.co/blog/javascript-framework-trends-in-2026-what-s-new-in-react-next.js-vue-angular-and-svelte)

**4. React Compiler 加速普及，useMemo/useCallback 逐渐成历史**
React Compiler 在 v1.0（2025年10月）后持续扩大采用率，自动处理记忆化与性能优化，手动使用 useMemo、useCallback 和 React.memo 被视为过时写法，开发者工作流大幅简化。
🔗 [LogRocket Blog](https://blog.logrocket.com/8-trends-web-dev-2026/)

**5. State of JavaScript 2025 调查结果：TypeScript 独占使用率升至 40%**
State of JS 2025 年度调查显示，40% 的受访开发者现已完全使用 TypeScript 编写代码（2024 年为 34%），超过 80% 的前端职位要求 TypeScript，其已成为专业开发的默认标准。
🔗 [InfoQ](https://www.infoq.com/news/2026/03/state-of-js-survey-2025/)

**6. Svelte March 2026 更新：程序化 createContext、HTML 注释、Laravel Starter Kit**
Svelte 官方 3 月更新包含程序化 Svelte 中的 createContext API、HTML 标签内注释支持，以及 Laravel + Svelte 官方 Starter Kit 和 warpkit（独立 Svelte 5 SPA 框架）等社区项目。
🔗 [Svelte Blog](https://svelte.dev/blog/whats-new-in-svelte-march-2026)

**7. Angular 21 Signals 减少 18% bundle 体积，响应式架构成主流**
Angular 21 的 Signals 架构在不依赖 Zone.js 的细粒度响应式模型下，使整体 bundle 体积降低约 18%，Angular 正在向更轻量、更现代的组件模型全面转型。
🔗 [Syncfusion Blog](https://www.syncfusion.com/blogs/post/frontend-development-trends)

**8. Cursor 3月更新：扩展 JetBrains 生态支持，Agent 能力大幅提升**
Cursor 3 月更新通过 Agent Client Protocol 实现对 IntelliJ、PyCharm、WebStorm 的原生集成，支持在 JetBrains IDE 中直接调用 OpenAI、Anthropic、Google 或 Cursor 模型进行 Agent 编程。
🔗 [The Agency Journal](https://theagencyjournal.com/cursors-march-2026-updates-jetbrains-integration-and-smarter-agents/)

**9. WebAssembly 成熟落地，Web 高性能应用开发进入新阶段**
2026 年 WebAssembly 已大规模落地，二进制编译带来更短的加载时间和更高的运行时性能，与 JavaScript 互操作能力持续增强，正成为浏览器端密集计算任务的首选方案。
🔗 [DEV Community](https://dev.to/blarzhernandez/what-will-shape-the-next-wave-of-frontend-development-in-2026-backed-by-experts-data-52h3)

**10. Perplexity 官方 MCP 服务器支持一键配置接入 Cursor IDE**
Perplexity 官方 MCP 服务器现已支持在 Cursor 中一键配置，开发者可直接在 IDE 内调用 Perplexity 的联网搜索和 Deep Research 能力，进一步增强 AI 辅助编程的上下文获取能力。
🔗 [Low Code Agency](https://www.lowcode.agency/blog/cursor-ide-perplexity-integration)
