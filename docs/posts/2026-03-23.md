## AI 资讯

**1. OpenAI GPT-5.4 mini / nano 发布：最快轻量级模型，向免费用户开放**
OpenAI 推出 GPT-5.4 mini 和 nano，在编码、推理与多模态理解上显著超越 GPT-5 mini，运行速度提升逾 2 倍；GPT-5.4 mini 已通过"Thinking"功能向 Free/Go 用户开放，GPT-5.1 系列同步于 3 月 11 日正式下线。
🔗 [OpenAI 官方公告](https://openai.com/index/introducing-gpt-5-4-mini-and-nano/)

**2. GPT-5.3-Codex：OpenAI 最强自治编码智能体上线**
GPT-5.3-Codex 融合 Codex 与 GPT-5 训练栈，定位为高度自治的 Agentic 编码模型，可自主规划并执行多步骤软件工程任务，在 OSWorld-V 桌面生产力基准测试中达到 75% 得分。
🔗 [OpenAI Academy](https://academy.openai.com/public/resources/latest-model)

**3. Anthropic Claude Code Channels：接入 Telegram 与 Discord 远程编程**
Anthropic 正式推出 Claude Code Channels 功能，允许开发者将 Claude Code 绑定至 Discord 或 Telegram，随时随地通过手机发送指令并接收完成通知，实现异步委托式代码开发。
🔗 [VentureBeat 报道](https://venturebeat.com/orchestration/anthropic-just-shipped-an-openclaw-killer-called-claude-code-channels)

**4. Anthropic 3 月双倍提升使用配额，Claude Sonnet 4.6 支持内联可视化**
Anthropic 宣布 3 月期间全线提升使用上限。Claude Sonnet 4.6 新增内联图表与可视化生成能力，支持 100 万 Token 上下文窗口（Beta），Agentic 搜索性能进一步提升。
🔗 [Analytics Insight](https://www.analyticsinsight.net/artificial-intelligence/anthropic-rolls-out-double-limits-for-claude-ai-check-timings-and-eligibility)

**5. Google Gemini 3.1 Flash-Lite：2.5 倍速度，超低价格**
Google 推出 Gemini 3.1 Flash-Lite，响应速度较前代提升 2.5 倍，输出生成速度提升 45%，每百万 token 定价仅 $0.25，成为高吞吐量低成本场景的首选。目前 Gemini 3.1 Pro 与 GPT-5.4 Pro 并列 AI 综合能力榜首，得分均为 57 分。
🔗 [AI 模型发布综述](https://renovateqr.com/blog/ai-model-releases-2026)

**6. DeepSeek V4 多模态版即将发布，适配华为/寒武纪国产芯片**
中国 DeepSeek 即将发布 V4 模型，支持图像、视频与文本生成的多模态能力，已与华为、寒武纪完成芯片适配优化，为国产硬件生态进一步提供 AI 基础设施支撑。
🔗 [PYMNTS 报道](https://www.pymnts.com/artificial-intelligence-2/2026/deepseek-poised-to-unveil-latest-ai-model/)

**7. Meta Llama 4 开源：MoE 架构达到 GPT-4.5 性能的 89%**
Meta 于 2026 年初发布 Llama 4，采用混合专家（MoE）架构，在关键基准达到 GPT-4.5 性能的 89%，支持消费级硬件全参数微调，进一步巩固开源 LLM 生态领导地位。
🔗 [DeepSeek vs Llama 2026](https://elephas.app/blog/deepseek-vs-llama-2025-comparison-which-local-ai-model-is-best-cm7kddany00ekip0lidqqoeq3)

**8. xAI Grok 4.20 正式上线，四代理协作架构突破多任务并行**
xAI Grok 4.20 引入独特的四代理协作架构，允许多个 AI 子智能体并行分工处理复杂任务。xAI 同步完成 200 亿美元 E 轮超额融资，Colossus 超算集群持续扩张。
🔗 [March 2026 AI 浪潮分析](https://integratedcognition.com/blog/march-2026s-ai-launch-wave-what-lawyers-should-make-of-gpt-54-claude-sonnet-46-gemini-31-pro-grok-420-glm-5-minimax-m25-and-the-deepseek-question)

**9. OpenAI 完成史上最大私募融资 1100 亿美元，估值达 7300 亿**
OpenAI 于 2 月 27 日宣布完成 1100 亿美元融资，亚马逊出资 500 亿，英伟达与软银各出资 300 亿，创私募融资历史纪录，彰显 AGI 赛道资本热度持续高涨。

加拿大研究团队发现，四大主流 AI 在使用新闻内容时几乎从不注明原始来源，ChatGPT 表现最差——即使覆盖了 54% 的特定报道内容，也鲜少标注原始媒体机构。
🔗 [Nieman Journalism Lab](https://www.niemanlab.org/2026/03/chatgpt-claude-gemini-and-grok-are-all-bad-at-crediting-news-outlets-but-chatgpt-is-the-worst-at-least-in-this-study/)

## 前端资讯

**1. State of JavaScript 2025：TypeScript 独占使用率升至 40%，TypeScript 已全胜**
State of JS 2025 调查（2 月发布）显示，40% 开发者仅使用 TypeScript（2024 年为 34%），80% 以上前端职位要求 TypeScript；Vite 满意度高达 98%，Astro 正向 Next.js 发起挑战。
🔗 [InfoQ 报道](https://www.infoq.com/news/2026/03/state-of-js-survey-2025/)

**2. Next.js 16 正式发布：Turbopack 默认启用，React Compiler 1.0 稳定集成**
Next.js 16 完整支持 React 19.2，Turbopack 在 `next dev` 和 `next build` 中默认开启；`use cache` 指令带来更显式的缓存控制；React Compiler 1.0 集成稳定，自动 Memoization 大幅减少不必要的重渲染。
🔗 [Next.js 官方博客](https://nextjs.org/blog/next-16)

**3. Vite 7 发布：五周年里程碑，最低支持 Node.js 20，默认目标升级**
Vite 7 标志着项目成立五周年，正式放弃 Node.js 18（已 EOL），默认浏览器目标切换为 `baseline-widely-available`，每个主版本随浏览器标准滚动更新，持续引领前端构建工具生态。
🔗 [Vite 7 公告](https://vite.dev/blog/announcing-vite7)

**4. React 19.2 发布：View Transitions、useEffectEvent 与 Activity 正式可用**
React 19.2 带来原生 View Transitions API 集成、`useEffectEvent` Hook 正式化以及 Activity 功能，与 Next.js 16 深度协同，为流畅页面过渡动画与后台任务管理提供框架级支持。
🔗 [Next.js 升级指南](https://nextjs.org/docs/app/guides/upgrading/version-16)

**5. Svelte 3 月更新：createContext API、HTML 注释，社区生态持续丰富**
Svelte 3 月更新新增程序化 `createContext` API、HTML 标签内注释支持，CSS 哈希值改为由 Svelte 自动稳定生成；State of JS 2025 中 Svelte 在响应式框架中好评度排名第一。
🔗 [Svelte 官方博客](https://svelte.dev/blog/whats-new-in-svelte-march-2026)

**6. React 市场份额 83.6% 但满意度承压，Next.js 负面评价创新高**
State of JS 2025 显示 React 使用率稳定在 83.6%，Next.js 使用率 59%，但 Next.js 负面评论量居所有工具之首（17% 负面 vs 21% 正面），开发者主要诉求集中在复杂度与文档质量。
🔗 [React vs Vue 2026](https://www.pkgpulse.com/blog/react-vs-vue-2026)

**7. Vue 3.6 Vapor Mode 极速挂载演示：彻底告别虚拟 DOM**
Vue 3.6 中的 Vapor Mode 完整 Demo 发布，通过跳过虚拟 DOM 层实现极致渲染性能，在大规模列表与动画密集场景下效果显著，是 Vue 生态重要的技术里程碑。
🔗 [JavaScript 框架趋势 2026](https://www.nucamp.co/blog/javascript-framework-trends-in-2026-what-s-new-in-react-next.js-vue-angular-and-svelte)

**8. TypeScript 5.9 发布：更智能的默认 tsconfig 生成**
TypeScript 5.9 优化了初始化配置生成逻辑，产出更简洁、贴近现代最佳实践的 `tsconfig.json` 默认配置，进一步降低新项目上手门槛，在 Node.js 稳定版中已支持原生类型剥离（Type Stripping）。
🔗 [前端开发趋势 2026](https://www.syncfusion.com/blogs/post/frontend-development-trends)

**9. Astro 启动基于 Oxc 的 Rust 编译器，向 Next.js 发起挑战**
Astro 团队透露正在研发基于 Oxc（Rust 工具链）的全新编译器，大幅提升构建性能和 DX；同期 Astro 已在满意度上超越 Next.js，成为 Meta 框架竞争格局的新变量。
🔗 [VoidZero 月报](https://voidzero.dev/posts/whats-new-feb-2026)

**10. AI 辅助前端开发成主流：Copilot 补全整组件，v0 一句话生成 UI**
GitHub Copilot 已可自动补全整个组件，Vercel v0 可从一行 Prompt 生成完整 Next.js UI，Remix 正向"AI-First"框架演进。AI 辅助工作流正在重塑前端开发范式，框架设计开始围绕 AI 协作展开。
🔗 [前端框架 2026 指南](https://www.kaashivinfotech.com/blog/frontend-development-frameworks-2026/)
