## AI 资讯

**1. MiniMax 发布 M2.7 自我进化模型**
MiniMax 于3月18日发布 M2.7 模型，具备自我进化能力，可自主完成30%-50%的强化学习研发流程，在 SWE-Pro 等基准测试中表现优异。定价仅 $0.30/百万输入 token，性价比极高。
🔗 [VentureBeat](https://venturebeat.com/technology/new-minimax-m2-7-proprietary-ai-model-is-self-evolving-and-can-perform-30-50)

**2. LangChain Deep Agents 3月大更新，GitHub 星标飙升**
LangChain Deep Agents 在3月进行重大更新，5小时内 GitHub 星标达 9.9k。同时宣布与 NVIDIA 战略合作，推出 AI-Q Blueprint 企业级智能体系统。
🔗 [MarkTechPost](https://www.marktechpost.com/2026/03/15/langchain-releases-deep-agents-a-structured-runtime-for-planning-memory-and-context-isolation-in-multi-step-ai-agents/)

**3. Perplexity 推出 Model Council 多模型协作功能**
Perplexity 3月上线 Model Council，同时调用 GPT-5.4、Claude Opus 4.6 和 Gemini 3.1 Pro 回答问题，展示模型间的共识与分歧。Deep Research 现基于 Opus 4.6 运行。
🔗 [Perplexity Changelog](https://www.perplexity.ai/changelog/what-we-shipped---march-6-2026)

**4. xAI 多位联合创始人离职，SpaceX 合并后动荡持续**
SpaceX 与 xAI 合并完成后，xAI 11位联合创始人中已有9人离开。Grok 因深度伪造内容问题面临多国监管调查。Grok 4.20 Beta 已在 API 平台上线。
🔗 [CNBC](https://www.cnbc.com/2026/02/11/musk-announces-xai-re-org-following-key-departures-spacex-merger.html)

**5. OpenAI 获 1100 亿美元融资，估值达 7300 亿**
OpenAI 从 Amazon、Nvidia 和 SoftBank 获得创纪录融资，估值达 7300 亿美元，进一步扩大 AWS 分发和 NVIDIA 基础设施能力。
🔗 [Crescendo AI](https://www.crescendo.ai/news/latest-ai-news-and-updates)

**6. 美国通过《AI 问责法案》**
美国于2026年3月通过 AI Accountability Act，要求在招聘、信贷、医疗、刑事司法等关键领域部署 AI 的公司定期发布偏见审计报告。
🔗 [Transparency Coalition](https://www.transparencycoalition.ai/news/ai-legislative-update-march20-2026)

**7. Block 裁员超4000人，CEO 称 AI 使小团队成为可能**
Jack Dorsey 领导的 Block 宣布裁员约40%（4000余人），表示 AI 工具已使更小、更高效的团队成为可能。
🔗 [AI News March 2026](https://theaitrack.com/ai-news-march-2026-in-depth-and-concise/)

**8. Cursor 3月更新：JetBrains 集成与 Agent 改进**
Cursor 通过 Agent Client Protocol 接入 IntelliJ、PyCharm 和 WebStorm，支持在 JetBrains IDE 中使用 OpenAI、Anthropic 等多家模型进行 Agent 编码。
🔗 [The Agency Journal](https://theagencyjournal.com/cursors-march-2026-updates-jetbrains-integration-and-smarter-agents/)

**9. Anthropic Claude Dispatch 功能上线 Max 订阅用户**
Claude Dispatch 允许 Max 用户在桌面端运行持久对话，通过手机远程发送消息，Claude 在本地完成任务后等待用户确认。
🔗 [Crescendo AI](https://www.crescendo.ai/news/latest-ai-news-and-updates)

**10. Meta 推出独立 AI 应用，搭载 Llama 4**
Meta 发布基于 Llama 4 的独立 AI 应用，支持语音交互、智能眼镜同步和社交发现信息流，正面对标 ChatGPT。
🔗 [Dan Shimony](https://www.danshimony.com/ai-race-as-of-today-march-20-2026/)

## 前端资讯

**1. Next.js 16.2 发布：启动速度提升400%**
Next.js 16.2 于3月18日发布，next dev 启动速度提升约400%，渲染速度提升50%。新增 AI 相关功能：AGENTS.md 脚手架、浏览器日志转发和实验性 Agent DevTools。
🔗 [Releasebot](https://releasebot.io/updates/vercel/next-js)

**2. Vite 8 正式发布：Rolldown 替代 Rollup**
Vite 8 于3月12日发布，核心变化是 Rust 编写的 Rolldown 替代 Rollup 作为生产打包器。实际项目构建速度提升高达87%（Linear 代码库从46秒降至6秒）。
🔗 [Syntackle](https://syntackle.com/blog/vite-7-is-here/)

**3. Svelte 3月更新：原生 select 样式与 CSS 解析器导出**
Svelte 5.47-5.49 版本支持使用 CSS 自定义 select 元素样式，CSS 解析器 parseCss 从 svelte/compiler 导出，Shadow DOM 自定义元素获得增强。
🔗 [Svelte Blog](https://svelte.dev/blog/whats-new-in-svelte-march-2026)

**4. Chrome 134 支持原生 select 元素样式化**
Chrome 134 带来了期待已久的 select 元素原生样式能力，包括新的 appearance 属性、伪元素和 selectedcontent 元素。
🔗 [Chrome for Developers](https://developer.chrome.com/docs/css-ui/anchor-positioning-api)

**5. Interop 2026 公布：20个焦点领域**
浏览器厂商联合推进 Interop 2026 计划，涵盖20个焦点领域（15个新增+5个延续），包括 Anchor Positioning 规范完善和更多跨浏览器兼容性改进。
🔗 [CSS-Tricks](https://css-tricks.com/interop-2026/)

**6. Safari 26.2 增强 Anchor Positioning 支持**
Safari 26.2 新增 flip-x 和 flip-y 选项，允许锚定元素在溢出容器时沿轴翻转，进一步完善 CSS Anchor Positioning 实现。
🔗 [WebKit Blog](https://webkit.org/blog/17640/webkit-features-for-safari-26-2/)

**7. Laravel 官方推出 Svelte Starter Kit**
Laravel 发布官方 Svelte 启动套件，基于 Inertia 提供 Laravel + Svelte 全栈开发的现代化起点。
🔗 [Svelte Blog](https://svelte.dev/blog/whats-new-in-svelte-march-2026)

**8. 新前端框架 Ripple 融合 React 与 Svelte**
Svelte 维护者 Dominic Gannaway 创建的 Ripple 框架，结合 React、SolidJS 和 Svelte 理念，采用 TypeScript 优先、JSX 编译、细粒度响应式和作用域 CSS。
🔗 [InfoQ](https://www.infoq.com/news/2025/12/ripple-front-end-framework-open/)

**9. React Compiler 进入 Release Candidate 阶段**
React Compiler 可自动分析 Hooks 使用并优化组件，消除不必要的重渲染，性能提升25-40%。Server Components 可将初始渲染时间从约2.4秒降至0.8秒。
🔗 [Netguru](https://www.netguru.com/blog/react-js-trends)

**10. TypeScript 成为前端开发事实标准**
2026年超80%前端职位要求 TypeScript 技能，纯 JavaScript 开发被视为遗留方式。端到端类型安全成为行业基线。
🔗 [Nucamp](https://www.nucamp.co/blog/javascript-framework-trends-in-2026-what-s-new-in-react-next.js-vue-angular-and-svelte)
