## AI 资讯

**1. Anthropic Claude 正式推出 Mac 电脑控制功能**
Claude 现可在 macOS 上为 Pro/Max 订阅用户自主操控电脑——包括移动鼠标、键盘输入、打开与编辑文件、提交 PR、运行测试等。支持通过 iOS 的 Dispatch 功能远程分配任务，基于"先请求权限"的安全设计。目前仅限 Mac，仍为研究预览版。
🔗 [9to5Mac](https://9to5mac.com/2026/03/23/anthropic-is-giving-claude-the-ability-to-use-your-mac-for-you/)

**2. Google Gemini AI 代理每日扫描千万条暗网帖子，识别企业安全威胁**
Gemini AI 代理被正式部署用于暗网监控，每天处理超 1000 万条帖子，针对特定组织提取潜在威胁情报，将 AI 能力扩展至主动安全防御领域。
🔗 [Let's Data Science](https://letsdatascience.com/news/gemini-agents-crawl-dark-web-for-threats-2de4ff61)

**3. Anthropic 发布 Claude Code Channels：可通过 Telegram/Discord 直接交互**
Claude Code Channels 允许用户将 Claude Code 接入自己的 Discord 或 Telegram，随时随地发消息驱动代码任务，Anthropic 定位为"OpenClaw 的直接竞争者"。
🔗 [VentureBeat](https://venturebeat.com/orchestration/anthropic-just-shipped-an-openclaw-killer-called-claude-code-channels)

**4. OpenAI GPT-5.4 mini 向免费用户开放，旧版深度研究模式 3/26 下线**
GPT-5.4 mini 通过"Thinking"功能面向 Free/Go 用户推出，并将于 3 月 26 日移除旧版 deep research 模式；同期更新 GPT-5.3 Instant 减少"预告式"语气回应。
🔗 [OpenAI Release Notes](https://help.openai.com/en/articles/6825453-chatgpt-release-notes)

**5. Elon Musk 宣布 Terafab：Tesla/SpaceX/xAI 合资建 240 亿美元芯片厂**
Terafab 将落地德克萨斯州奥斯汀，生产面向电动汽车、Optimus 机器人及高性能 AI 计算的定制芯片，是迄今美国最大的单体 AI 硬件投资项目之一。
🔗 [TechStartups](https://techstartups.com/2026/03/23/top-tech-news-today-march-23-2026/)

**6. OpenAI 年化收入破 250 亿美元，计划 2026 年底员工翻倍并筹备 IPO**
内部文件显示 OpenAI 计划将员工规模从约 4000 人扩至 8000 人，重点投入工程、安全与企业销售；公司也在就潜在 IPO 时间线进行早期评估。
🔗 [TechStartups](https://techstartups.com/2026/03/23/top-tech-news-today-march-23-2026/)

**7. OpenAI 将向 ChatGPT 免费/Go 用户展示广告**
ChatGPT 宣布即将在美国免费版和 Go 版用户中引入广告，作为收入多元化举措；这是 OpenAI 首次在消费端引入广告模式，理由是算力成本持续攀升。
🔗 [LLM Stats](https://llm-stats.com/ai-news)

**8. Perplexity 推出 Personal Computer：Mac mini 上运行的 24/7 本地 AI 代理**
Perplexity Computer 将本地文件、应用与会话融合，作为数字代理持续在后台执行主动任务、监控触发条件，并同步推出 Slack 工作流和企业级升级。
🔗 [Releasebot](https://releasebot.io/updates/perplexity-ai)

**9. Nvidia Vera Rubin 平台发布：推理吞吐量提升 10 倍**
GTC 2026 上 Jensen Huang 发布 Vera Rubin 平台，称其为"历史上最大规模基础设施建设的起点"；AWS、Google Cloud、Azure、Oracle Cloud 均已确认将部署该平台。
🔗 [VentureBeat](https://venturebeat.com/)

**10. 特朗普政府发布国家 AI 立法框架：联邦统一监管**
白宫发布 AI 立法建议框架，涵盖儿童保护、知识产权、言论自由等七大领域；核心诉求是由国会立联邦统一法规，预先取代各州日益增多的 AI 监管条例。
🔗 [White House](https://www.whitehouse.gov/articles/2026/03/president-donald-j-trump-unveils-national-ai-legislative-framework/)

## 前端资讯

**1. Node.js 今日发布多版本安全补丁（25.x/24.x/22.x/20.x）**
Node.js 官方于 3 月 24 日发布所有主要维护版本的安全更新，修复 2 个高危漏洞及多个中危漏洞，强烈建议生产环境尽快升级。
🔗 [nodejs.org](https://nodejs.org/en/blog/vulnerability/march-2026-security-releases)

**2. Next.js 16.2 发布：AI 代理就绪、dev 启动速度提升 87%**
新版带来 Agent-ready CLI 脚手架、浏览器日志转发至终端、实验性 Agent DevTools，以及 Turbopack 200+ bug 修复；生产构建提速显著。
🔗 [Next.js Blog](https://nextjs.org/blog/next-16-2)

**3. State of JavaScript 2025 调查结果出炉：TypeScript "已经赢了"**
40% 开发者现在只写 TypeScript（2022 年为 28%）；Vite 满意度达 98% 创历史新高；React 以 44.7% 使用率继续领跑，Webpack 满意度持续下滑。
🔗 [InfoQ](https://www.infoq.com/news/2026/03/state-of-js-survey-2025/)

**4. Webpack 2026 路线图：原生 CSS 模块 + 内置 TypeScript，迈向 v6**
Webpack 团队发布详细路线图，计划将 CSS Modules、TypeScript 编译和 HTML 入口支持内置到核心，消除对 mini-css-extract-plugin、ts-loader 等插件的依赖。
🔗 [InfoQ](https://www.infoq.com/news/2026/03/webpack-2026-roadmap/)

**5. Interop 2026 正式启动：三大浏览器引擎联合攻克 CSS 跨浏览器兼容**
Chrome/Edge（Blink）、Safari（WebKit）、Firefox（Gecko）第五年联手，重点推进 CSS Anchor Positioning、Scroll-Driven Animations、Cross-Document View Transitions 等特性的一致实现。
🔗 [web.dev](https://web.dev/blog/interop-2026)

**6. Chrome 137 引入 CSS if() 函数，原生条件样式无需 JavaScript**
开发者可在 CSS 中直接写条件样式逻辑，无需借助 JS 或 CSS-in-JS 方案；标志着 CSS 能力持续向传统 JS 领域延伸。
🔗 [LogRocket](https://blog.logrocket.com/css-in-2026/)

**7. ESLint 正式支持 CSS 语言检查，统一全语言规范工具链**
ESLint v9.15.0+ 发布官方 CSS 语言插件，一套配置即可覆盖 JS/CSS/JSON/Markdown，适配新版 flat config 系统。
🔗 [ESLint CSS GitHub](https://github.com/eslint/css)

**8. Cloudflare 开源 vinext：AI 一周重写 Next.js，构建速度快 4.4x**
vinext 是基于 Vite 实现的 Next.js 开源替代品，覆盖 94% Next.js 16 API，33 路由应用生产构建仅需 1.67 秒（Next.js 为 7.38 秒），几乎全部代码由 Claude AI 编写完成。
🔗 [Cloudflare Blog](https://blog.cloudflare.com/vinext/)

**9. Bun 1.3 发布：支持直接运行 HTML 文件，全栈开发零配置**
Bun 现可将 HTML 文件作为入口点直接执行，内置 React/CSS/JS 打包；同时新增统一数据库 API 和 Redis 支持。
🔗 [Bun Blog](https://bun.com/blog/bun-v1.3)

**10. 欧洲无障碍法案（EAA）将于 6 月 28 日强制生效**
凡面向欧盟市场的数字产品须满足 WCAG 无障碍标准，违规面临罚款；前端开发者应尽早审计键盘导航、对比度、ARIA 标注等问题。
🔗 [LogRocket](https://blog.logrocket.com/8-trends-web-dev-2026/)
