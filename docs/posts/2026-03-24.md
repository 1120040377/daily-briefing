## AI 资讯

**1. Anthropic Claude 正式登陆 Mac，可自主操控电脑完成任务**
Anthropic 于 3 月 23 日发布 Claude Computer Use for Mac（研究预览版），支持在 Claude Pro/Max 订阅中使用，可移动鼠标、键盘输入、打开 App、填写表格，并通过 Dispatch 从手机远程下发任务。
🔗 [9to5Mac](https://9to5mac.com/2026/03/23/anthropic-is-giving-claude-the-ability-to-use-your-mac-for-you/)

**2. Anthropic Claude Code Channels：接入 Discord 与 Telegram**
Claude Code Channels 允许用户将 Claude Code 连接到 Discord 或 Telegram，随时随地通过消息与 Claude Code 互动，在外出时也能下发代码任务并查看进度。
🔗 [VentureBeat](https://venturebeat.com/orchestration/anthropic-just-shipped-an-openclaw-killer-called-claude-code-channels)

**3. 苹果官宣 WWDC 2026 将展示重磅 AI 新功能**
苹果于 3 月 23 日公布 WWDC 2026 日期（6 月 8-12 日），明确预告将发布"AI 全面进化"内容，包括可与 ChatGPT、Gemini、Claude 竞争的全新 Siri 聊天机器人版本。
🔗 [9to5Mac](https://9to5mac.com/2026/03/23/apple-promises-ai-advancements-to-be-unveiled-at-wwdc-this-year/)

**4. Cursor 发布 Composer 2：AI 自主处理跨文件大型编码任务**
Cursor 发布 Composer 2，能够自主规划并跨整个代码库执行多文件复杂任务，用户只需描述需求，Agent 即可自行制定方案并执行。
🔗 [AI Releases](https://www.labla.org/latest-ai-model-releases-past-24-hours/new-ai-tools-model-updates-march-23-2026-the-companies-you-havent-heard-of-yet/)

**5. Meta 推出 AI 自动投放广告 Agent**
Meta 宣布推出 AI Agents，可在 Ads Manager 和 WhatsApp Business 中全程自主完成广告活动的创建、优化和投放，无需人工干预。
🔗 [The Prompt Report](https://www.wbn.digital/the-prompt-report-march-23-2026-the-ai-land-grab-has-begun/)

**6. Elon Musk 宣布 Terafab：Tesla/SpaceX/xAI 合资建 240 亿美元芯片厂**
Terafab 将落地德克萨斯州奥斯汀，生产面向电动汽车、Optimus 机器人及高性能 AI 计算的定制芯片，是美国最大的单体 AI 硬件投资项目之一。
🔗 [TechStartups](https://techstartups.com/2026/03/23/top-tech-news-today-march-23-2026/)

**7. OpenAI 将向 ChatGPT 免费/Go 用户展示广告**
ChatGPT 宣布即将在美国免费版和 Go 版用户中引入广告，这是 OpenAI 首次在消费端引入广告模式，理由是算力成本持续攀升。
🔗 [LLM Stats](https://llm-stats.com/ai-news)

**8. Perplexity 推出 Personal Computer：Mac mini 上运行的 24/7 本地 AI 代理**
Perplexity Computer 将本地文件、应用与会话融合，作为数字代理持续在后台执行主动任务、监控触发条件，并同步推出 Slack 工作流和企业级升级。
🔗 [Releasebot](https://releasebot.io/updates/perplexity-ai)

**9. Nvidia Vera Rubin 平台发布：推理吞吐量提升 10 倍**
GTC 2026 上 Jensen Huang 发布 Vera Rubin 平台，AWS、Google Cloud、Azure、Oracle Cloud 均已确认将部署该平台。
🔗 [VentureBeat](https://venturebeat.com/)

**10. 特朗普发布国家 AI 立法框架，同步推进 291 页 TRUMP AI 法案**
白宫正式公布国家 AI 立法框架；同期参议员提交 291 页《TRUMP America AI Act》，拟建立美国首个全面联邦 AI 监管体系，涵盖儿童保护、知识产权、AI 风险与责任等领域。
🔗 [白宫](https://www.whitehouse.gov/articles/2026/03/president-donald-j-trump-unveils-national-ai-legislative-framework/)

## 前端资讯

**1. Vite 8 正式发布：Rolldown 统一打包器，构建速度提升 10-30 倍**
Vite 8 以 Rust 编写的 Rolldown 取代原有的 esbuild + Rollup 双引擎架构，实现单一统一打包器，同步推出插件目录 registry.vite.dev 和集成 DevTools。
🔗 [Vite 官博](https://vite.dev/blog/announcing-vite8)

**2. Next.js 16.2 发布：启动速度提升 400%，内置 AI Agent 支持**
新版带来 Agent-ready CLI 脚手架、浏览器日志转发至终端、实验性 Agent DevTools，以及 Turbopack 200+ bug 修复。
🔗 [Next.js Blog](https://nextjs.org/blog/next-16-2)

**3. Node.js 多版本安全补丁发布（25.x/24.x/22.x/20.x）**
Node.js 官方于 3 月 24 日发布所有主要维护版本的安全更新，修复 2 个高危漏洞及多个中危漏洞，建议生产环境尽快升级。
🔗 [nodejs.org](https://nodejs.org/en/blog/vulnerability/march-2026-security-releases)

**4. Interop 2026 启动：三大浏览器引擎联合攻克 CSS 跨浏览器兼容**
Chrome、Safari、Firefox 联手推进 20 项互操作性目标，涵盖 CSS 锚点定位、滚动驱动动画、容器样式查询、跨文档 View Transitions 等。
🔗 [CSS-Tricks](https://css-tricks.com/interop-2026/)

**5. Chrome 137 引入 CSS if() 函数，原生条件样式无需 JavaScript**
开发者可在 CSS 中直接写条件样式逻辑，标志着 CSS 能力持续向传统 JS 领域延伸。
🔗 [LogRocket](https://blog.logrocket.com/css-in-2026/)

**6. Cloudflare 开源 vinext：AI 一周重写 Next.js，构建速度快 4.4 倍**
vinext 覆盖 94% Next.js 16 API，33 路由应用生产构建仅需 1.67 秒（Next.js 为 7.38 秒），几乎全部代码由 Claude AI 编写完成。
🔗 [Cloudflare Blog](https://blog.cloudflare.com/vinext/)

**7. CSS 原生 Masonry 布局与 Mixin 即将跨浏览器落地**
display: masonry 原生瀑布流、@mixin/@apply 原生 CSS Mixin、sibling-index()/sibling-count() 函数预计年底前在主流浏览器稳定支持。
🔗 [LogRocket](https://blog.logrocket.com/css-in-2026/)

**8. Webpack 2026 路线图：原生 CSS 模块 + 内置 TypeScript，迈向 v6**
计划将 CSS Modules、TypeScript 编译和 HTML 入口支持内置到核心，消除对外部 loader/plugin 的依赖。
🔗 [InfoQ](https://www.infoq.com/news/2026/03/webpack-2026-roadmap/)

**9. Bun 1.3 发布：支持直接运行 HTML 文件，全栈开发零配置**
Bun 可将 HTML 文件作为入口点直接执行，内置 React/CSS/JS 打包，同时新增统一数据库 API 和 Redis 支持。
🔗 [Bun Blog](https://bun.com/blog/bun-v1.3)

**10. 欧洲无障碍法案（EAA）将于 6 月 28 日强制生效**
面向欧盟市场的数字产品须满足 WCAG 无障碍标准，前端开发者应尽早审计键盘导航、对比度、ARIA 标注等问题。
🔗 [LogRocket](https://blog.logrocket.com/8-trends-web-dev-2026/)
