## AI 资讯

**1. OpenAI 正式关闭 Sora 视频应用，迪士尼同步终止 10 亿美元投资**
OpenAI 于 3 月 24 日宣布关闭上线不足一年的 Sora 短视频 App，官方表示将计算资源转向更高收益的编程与推理任务。迪士尼也随即宣布终止与 OpenAI 计划中的 10 亿美元战略合作。
🔗 [TechCrunch](https://techcrunch.com/2026/03/24/openais-sora-was-the-creepiest-app-on-your-phone-now-its-shutting-down/)

**2. Anthropic Claude Code 推出 Auto Mode：AI 自主批准操作，告别繁琐确认**
Claude Code 新增 Auto Mode 权限功能，可由 AI 自主判断每个操作是否安全并直接执行，同时内置安全分类器监控提示注入风险。目前仅限 Team 计划管理员开启，Enterprise/API 支持即将跟进。
🔗 [TechCrunch](https://techcrunch.com/2026/03/24/anthropic-hands-claude-code-more-control-but-keeps-it-on-a-leash/)

**3. 联邦法官公开质问五角大楼：将 Anthropic Claude 列入黑名单有何依据**
法官 Rita Lin 在听证会上指出，国防部对 Anthropic Claude 的供应链风险认定"看起来像是试图重创公司"。Anthropic 申请临时叫停五角大楼黑名单及特朗普禁令，案件持续受到关注。
🔗 [CNBC](https://www.cnbc.com/2026/03/24/anthropic-lawsuit-pentagon-supply-chain-risk-claude.html)

**4. Gap 成首家在 Google Gemini 内实现原生购物结账的时尚品牌**
用户在 Gemini 中搜索 Gap 旗下品牌商品时，可直接完成购买，无需跳转至品牌官网。这是 Gemini Commerce 商业化的重要里程碑，标志着 AI 平台正式切入零售闭环。
🔗 [CNBC](https://www.cnbc.com/2026/03/24/gap-google-gemini-checkout-ai-platform.html)

**5. 中国 AI 股大涨：官媒强调国内大模型 Token 使用量激增**
3 月 25 日，官方媒体报道国内 AI 模型采用率和 Token 消耗量大幅上升，智谱 AI 在港股单日涨幅达 8%，MiniMax 等同类企业股价也随之拉升。
🔗 [Bloomberg](https://www.bloomberg.com/news/articles/2026-03-25/china-s-ai-stocks-gain-as-state-media-tout-rising-token-usage)

**6. ChatGPT GPT-5.4 mini 全面推送，大学生可领 100 美元 Codex 积分**
OpenAI 正在向 Free/Go/Plus 用户全量推送 GPT-5.4 mini，同时面向美国和加拿大经认证的大学生开放 Codex 服务，每人赠送价值 100 美元（约 2500 积分）的额外用量。
🔗 [Releasebot](https://releasebot.io/updates/openai/chatgpt)

**7. OpenAI 与美国能源部深化 AI 合作，聚焦能源基础设施与科学研究**
OpenAI 官宣与美国能源部扩大合作范围，涵盖核能、电网管理及科学研究领域，这是 OpenAI 加深与联邦政府机构绑定的最新举措。
🔗 [OpenAI](https://openai.com/index/us-department-of-energy-collaboration/)

**8. ChatGPT 旧版"深度研究"模式将于 3 月 26 日正式下线**
OpenAI 宣布明日起停止提供旧版 Deep Research 入口，现有对话记录不受影响，用户将自动迁移至当前最新的深度研究体验。
🔗 [Releasebot](https://releasebot.io/updates/openai/chatgpt)

**9. Claude Sonnet 4.6 发布：100 万 Token 上下文窗口（Beta），登顶编程排行榜**
Anthropic 推出目前最强的 Sonnet 系列模型 Claude Sonnet 4.6，在编程、Computer Use、长上下文推理、Agent 规划等维度全面升级，并以 Arena 分 1051 领跑 LLM 编程能力榜。
🔗 [LLM Stats](https://llm-stats.com/ai-news)

**10. Perplexity 为 Pro/Max 用户推出 GPT-5.2 深度推理功能**
Perplexity 近期为订阅用户集成了 GPT-5.2 模型，支持多步骤深度推理，进一步提升搜索问答与研究场景下的回答质量。
🔗 [Releasebot](https://releasebot.io/updates/perplexity-ai)

## 前端资讯

**1. TypeScript 6.0 正式发布：末代 JavaScript 版，为 Go 原生 7.0 铺路**
TypeScript 6.0 于 3 月 23-24 日正式发布，是最后一个基于 JavaScript 代码库的版本。新增内置 Temporal API 类型、Map Upsert 方法支持，并移除 classic 模块解析策略，为即将到来的 Go 编译器驱动的 TypeScript 7.0 做好兼容性过渡。
🔗 [Microsoft DevBlogs](https://devblogs.microsoft.com/typescript/announcing-typescript-6-0/)

**2. Node.js 多版本安全补丁发布（25.x / 24.x / 22.x / 20.x）**
Node.js 于 3 月 24 日同步发布所有活跃维护版本的安全更新，修复 2 个高危漏洞（TLS SNI 回调崩溃、HTTP/2 内存泄漏）及 V8 哈希碰撞可预测问题，建议生产环境尽快升级。
🔗 [nodejs.org](https://nodejs.org/en/blog/vulnerability/march-2026-security-releases)

**3. Chrome 宣布 9 月起改为双周发布周期，Web 新特性落地更快**
Google 于本月初宣布 Chrome 将从 Chrome 153（9 月 8 日）起由 4 周发布周期切换为双周节奏，意味着 CSS/JS 新标准将更快进入用户手中，也要求开发者更频繁关注兼容性变化。
🔗 [Chrome for Developers](https://developer.chrome.com/blog/chrome-two-week-release)

**4. Chrome 146 Stable：CSS Carousel 原生组件、View Transitions 转正、@scope 稳定**
Chrome 146 带来原生 CSS 轮播组件（::scroll-button / ::scroll-marker）、View Transitions API 正式转正、@scope 规则稳定，以及 Invoker Commands 可声明式触发元素行为，无需 JavaScript。
🔗 [Chrome for Developers](https://developer.chrome.com/release-notes/146)

**5. Svelte "What's new in Svelte: March 2026" 月报出炉**
本月 Svelte 生态更新涵盖 SvelteKit Netlify Adapter 现代化升级、Svelte CLI 正式支持 better-auth 插件，以及 State of JS 2025 中 Svelte 蝉联响应式框架满意度榜首。
🔗 [Svelte Blog](https://svelte.dev/blog/whats-new-in-svelte-march-2026)

**6. InfoQ：State of JavaScript 2025 报告——TypeScript 称霸，Vite 满意度 98%**
13000+ 开发者参与的调查显示，TypeScript 已被 40% 开发者作为唯一语言使用，Vite 满意度高达 98%，React 使用率仍居首（83.6%）但负面反馈上升，AI 生成代码占比接近 30%。
🔗 [InfoQ](https://www.infoq.com/news/2026/03/state-of-js-survey-2025/)

**7. Next.js 16.2 发布：冷启动提速 400%，内置 AI Agent 开发工具链**
Next.js 16.2 带来 Agent-ready create-next-app 脚手架、浏览器日志转发至终端、实验性 Agent DevTools，以及 Turbopack 的 Server Fast Refresh 和 Web Worker Origin 支持，含 200+ Bug 修复。
🔗 [Next.js Blog](https://nextjs.org/blog/next-16-2)

**8. Bun v1.3.10 发布：原生 REPL、HTML 文件直接编译，性能大幅跃升**
Bun 1.3.10 新增内置交互式 REPL、--compile --target=browser 生成自包含 HTML，structuredClone 性能提升 25 倍，path.parse() 提升 7 倍，同时支持 TC39 标准 ES 装饰器和 Windows ARM64。
🔗 [Bun Blog](https://bun.com/blog)

**9. Angular 21.2.5 安全修复版本发布**
Angular 团队发布 21.2.5 补丁版本，修复若干安全和稳定性问题，Angular 22 仍计划于 2026 年 5 月发布并引入信号级别的全局优化。
🔗 [Angular Releases](https://angular.dev/reference/releases)

**10. QCon London 2026：在浏览器端直接运行 AI 推理，WebGPU 正成熟**
开发者分享了使用 Transformers.js + WebGPU 在浏览器本地运行 AI 工作负载的实践，覆盖隐私保护、低延迟与零服务器成本等优势场景，标志着端侧 AI 推理工具链日趋成熟。
